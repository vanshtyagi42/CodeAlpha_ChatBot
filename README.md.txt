# CodeAlpha_ChatBot:

# Project Description:
This is a friendly chatbot developed using Python.The chatbot uses regular expressions to understand user inputs and respond appropriately.

## Features:

- Greeting responses
- Date and time information
- Basic arithmetic calculations
- Jokes
- Help menu
- Exit commands

## Technologies Used:

- Python
- Regular Expressions (re)
- Random Module
- Datetime Module

## Sample Commands:

- hello
- how are you
- what is your name
- what is 5 + 3
- tell me a joke
- help
- bye
